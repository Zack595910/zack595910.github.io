📖 项目简介

KSD 是一个基于 Python Tkinter 构建的模块化冒险游戏框架。自 Core 1.0 起经历七个架构版本迭代，现已发展为功能完善的游戏平台。支持用户系统、背包系统、密钥管理、主题切换、Shell调试面板、模块动态加载等丰富功能。

项目由 闪讯工作社 Snxth. 开发维护，数据存储在 C:\Date\Snxth.\KSD Run Files\。

✨ 核心功能

🎮

经典玩法

WASD控制角色移动，收集金色经验球升级，每球+1金币，最高5级

👤

用户系统

注册/登录/记住密码，SHA256加密存储，管理员权限分离

🎒

背包系统

10种物品，支持购买/出售/清空，管理员免费获取，容量50格

🔑

密钥系统

6种验证方式（明文/哈希/Base64/JSON/主密钥/双重加密），密钥冻结与日志

🎨

主题切换

经典Windows / 深色模式 / 浅色模式，实时切换生效

💻

Shell面板

底部终端风格日志窗口，6种彩色标签，实时调试输出

📦

模块管理

内置模块管理器，导入/加载/卸载第三方模块，支持外部.py/.ini任务文件

📝

任务编辑器

4步向导式创建INI任务文件，预览后保存并导入测试

📷

截图系统

F12一键截图，保存为PostScript格式，存入Screenshots文件夹

💾

备份恢复

完整数据备份与恢复功能，支持JSON/CSV格式导出

🏗️ 架构版本

Core 1.0 → 最初版本，直接exec()执行模块
Core 2.0 → 增加用户系统、背包系统
Core 3.0 → 密钥系统、Shell面板、通知系统、主题切换
Core 4.0 → 模块类封装，stop_game_loops标记
Core 5.0 → safe_after自动追踪、bind_key自动解绑
Core 6.0 → 模块生命周期完善、向导式任务编辑器
Core 7.0 → 当前版本，主程序与核心模块分离，模块管理器内置，数据路径重构
